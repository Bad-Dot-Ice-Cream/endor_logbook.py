Upon running you will be greeted with:
"-- Endor Logbook --
1) Write entry
2) Random encounter
3) Read all
4) Search
5) Quit
> "         (User input space)

Enter the number "1" to write an entry that will be saved in the log.
Enter "2" to save a random event to the log.
Enter "3" to view all previous log entires, both random and personal. All time stamped and listed in order.
Eneter "4" to search for logs with a specific keyword.
Enter "5" to quit the program.
